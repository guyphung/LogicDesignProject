# LOGIC DESIGN PROJECT
## PREREQUISITES
* **Python 3.11** (Recommended for compatibility).
* **Webcam** (Built-in or USB).
* **Git** installed on your machine.

## INSTALLATION
### 1. Clone the repository
Open your terminal or command prompt and run:
```bash
git clone https://github.com/guyphung/LogicDesignProject.git
cd LogicDesignProject
```

### 2. Install All required libraries that are listed in requirements.txt
Open your terminal or command prompt and run:
```bash
python -m pip install -r requirements.txt
```

## HOW TO RUN
1. Ensure your webcam is connected.
2. Run the main script:
```bash
python main.py
```
3. Controls:
* The system will open a window showing the live feed.
* Press Q on your keyboard to exit the application.

## PRECAUTION
* Both the main application and the trained YOLOv5 model weights file MUST be in the same directory
* ALL dependencies listed in requirements.txt must be installed for the project to work
